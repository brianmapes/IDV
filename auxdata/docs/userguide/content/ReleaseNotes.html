The items below reflect changes since the 5.5 release.

The version of the netCDF-Java library currently distributed with the IDV is the 4.6.13. This is a bug fix release. See the netCDF-Java Library for a more details information.

Java Webstart, Windows app and MacOS certificates have been renewed and will be valid until at least May 30, 2020 (MacOS certificate is valid until 2024). Moreover, as properly signing the IDV under these different environments can be an involved process, this information has thoroughly documented here.

For synoptic scale motions, atmospheric variables tend to be better correlated along an isentropic surface upstream/downstream. In a 3D space, the new grid 3D isentropic advection derived parameter interpolates advection from isobaric coordinate into the isentropic space and creates 3D theta surface advection displays.

The GLM Level2 products consist of geo-located and time-stamped events, groups, and flashes, with associated calibrated optical amplitudes (in units of Joules). A GLM ADDE data chooser has been developed to access three GLM source types: Geostationary Lightning Mapper EVENT, GROUP, and FLASH Data through the ADDE data server. The GLM data queries can be performed on the relative times, absolute times, and time driver times

Streamlines are a group of curves that instantaneous tangent to the velocity vector of the flow and a snapshot of the flow field at some particular time. In the IDV 3D space, streamline display provides the direction in which a massless fluid element will travel at any point in time. 3D streamline display type is available for both derived parameters: Grid 3D Trajectory and 3D Flow Vectors, and the initial area of the 3D streamlines can be a 2D surface or a 3D volume.

The grid 3D trajectory display includes three new display types: Tracer Arrow, Tracer Sphere and Tracer Dot. The initial area of the 3D Grid Trajectory can also be set at XZ or YZ plane, and controlled by the data index in the X or Y direction.

New vertical profile display that uses pressure as its Y axis to emphasize information in the lower level atmoshpere and to understanding tropical convection - moisture interactions mainly concentrated in lower atmosphere. A new derived variable: Conserved Sounding (from theta, thetae and sat thetae), and its associated display Conserved Variable Sounding (Pressure) has been added to the derived parameter list of gridded dataset.

A collection of utilities from Mapes IDV collection performs a series of basic grid divide, multiply, modulo, and vertical derivative.

A collection of utilities from Mapes IDV collection applies a series of horizontal smoothing on 3D field.

A collection of utilities from Mapes IDV collection calculates a series of daily, monthly, yearly mean and anomaly.

The new ISL publish tag will publish a file to RAMADDA server.

We fixed a bug associated with the excessive data loading in this release. Another bug fixed is to handle the redirect of the http urls.

For a list of outstanding known problems, see the Known Problems page.
